## 功能
从[代理池](https://github.com/LogicJake/Proxy_IP)获取代理进行返回
## 接口说明
### 获取随机代理
| 名称 | 选项 |
|:---:|:---:|
| 接口地址 | /index.php?_action=getIP|
| 返回格式 | JSON
| 请求方法 | GET
| 请求示例 | http://api.logicjake.xyz/get_proxy/?_action=getIP&validcode=数据表user中自定义的验证码

#### 返回
```
{
    "code": 0,
    "data": {
        "IP": "163.125.20.136",
        "PORT": "9999"
    }
}
```
